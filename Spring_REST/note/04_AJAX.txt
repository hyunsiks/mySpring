# AJAX(Async JavaScript and XML)

 - 웹페이지 전체를 다시 로딩하지 않고, 웹 페이지의 일부분만 갱신하는 자바스크립트 문법
 - 웹페이지 백그라운드 영역에서 서버와 통신하여 데이터를 받아온 후 그 데이터를 활용한다
 - 원래는 XML을 받아오는 용도로 개발되었지만 최근에는 JSON을 받아오는 용도로 더 많이 사용
 
# JavaScript XMLHttpRequest

 - 서버로 비동기 요청을 보낼 수 있는 클래스
 - 요청을 보낸 후 서버의 처리 상황에 따라 readyState가 변한다
 - 서버에서 응답해준 상태 코드를 활용할 수 있다 (Spring의 RespenseEntity)
 - GET방식으로 전송할 때는 opne() 메서드의 URI뒤에 데이터를 추가한다
 - POST방식으로 데이터를 전송할 때는 send() 메서드의 매개변수에 데이터를 추가한다
